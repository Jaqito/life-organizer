# Instruction For Installation


##Technologies Required:

###Yarn or NPM

```
brew install yarn
```
OR
```$xslt
brew install npm
```
###Install packages
```
cd frontend && yarn
```

###API keys 
```$xslt
News API - https://newsapi.org/
Marketstack - https://marketstack.com/
```


###To start program

```
cd frontend && yarn start
```
```$xslt
Open localhost:3000
```
